## [Installation](https://stylelint.io/user-guide/get-started)

Using pnpm

```sh
pnpm add -D stylelint

# Install the related configs and plugins
pnpm add -D stylelint-a11y stylelint-config-standard stylelint-order stylelint-plugin-defensive-css stylelint-use-logical-spec
```

## Usage

and here is a script to run stlintlint on your project:

```sh
# File: package.json

"style:lint": "pnpm run stylelint --cache './src/**/*.css'",
"style:lint:fix": "pnpm run style:lint --fix",
```
