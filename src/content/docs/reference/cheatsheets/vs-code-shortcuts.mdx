## List of Useful Shortcuts

- File navigation: <Kbd mac="Command+P" windows="Control+P" linux="Control+P" />
  / <Kbd mac="Command+E" windows="Control+E" linux="Control+E" />

- Side panel navigation: <Kbd mac="Command+Shift+E" windows="Control+Shift+E"
  linux="Control+Shift+E" />

- Toggle side panel: <Kbd mac="Command+B" windows="Control+B" linux="Control+B" />

- Organize imports: <Kbd mac="Shift+Option+O" windows="Shift+Alt+O" linux="Shift+Alt+O" />

- Comment line: <Kbd mac="Command+:" windows="Control+:" linux="Control+:" /> (remapped to
  <Kbd mac="Shift+Command+:" windows="Shift+Control+:" linux="Shift+Control+:"
  />)

- Sort lines: <Kbd mac="Command+Option+S" windows="Control+Alt+S" linux="Control+Alt+S" />

- Terminal: <Kbd mac="Command+Ã¹" windows="Control+`" linux="Control+`" /> (remapped to
  <Kbd mac="Control+$" windows="Control+$" linux="Control+$" />)

- Multiple cursors: <Kbd mac="Command+Option+Up" windows="Control+Alt+Up" linux="Control+Alt+Up"
  /> / <Kbd mac="Command+Option+Down" windows="Control+Alt+Down"
  linux="Control+Alt+Down" />
  <Kbd mac="Control+Shift+Up" windows="Control+Shift+Up"
  linux="Control+Shift+Up" /> / <Kbd mac="Control+Shift+Down"
  windows="Control+Shift+Down" linux="Control+Shift+Down" />

- Search: <Kbd mac="Command+F" windows="Control+F" linux="Control+F" />

- Replace: <Kbd mac="Command+H" windows="Control+H" linux="Control+H" />

- File search: <Kbd mac="Command+Shift+F" windows="Control+Shift+F" linux="Control+Shift+F"
  /> / <Kbd mac="Command+Shift+H" windows="Control+Shift+H"
  linux="Control+Shift+H" />

- Duplicate up/down: <Kbd mac="Shift+Option+Up" windows="Shift+Alt+Up" linux="Shift+Alt+Up" /> /
  <Kbd mac="Shift+Option+Down" windows="Shift+Alt+Down" linux="Shift+Alt+Down"
  />

- Move line up/down once: <Kbd mac="Option+Up" windows="Alt+Up" linux="Alt+Up" /> / <Kbd
  mac="Option+Down" windows="Alt+Down" linux="Alt+Down" />

- Multiple cursors: <Kbd mac="Option+Click" windows="Alt+Click" linux="Alt+Click" />
  <Kbd mac="Command+Option+Up" windows="Control+Alt+Up" linux="Control+Alt+Up"
  /> / <Kbd mac="Command+Option+Down" windows="Control+Alt+Down"
  linux="Control+Alt+Down" />

- Go up / down: <Kbd mac="Command+Home" windows="Control+Home" linux="Control+Home" /> / <Kbd
  mac="Command+End" windows="Control+End" linux="Control+End" />

- Delete previous word: <Kbd mac="Option+Enter" windows="Alt+Enter" linux="Alt+Enter" />

- Delete/cut a line: <Kbd mac="Command+X" windows="Control+X" linux="Control+X" />

- Select entire words: <Kbd mac="Option+Shift+Left" windows="Alt+Shift+Left" linux="Alt+Shift+Left"
  /> / <Kbd mac="Option+Shift+Right" windows="Alt+Shift+Right"
  linux="Alt+Shift+Right" />

- Display Main Sidebar: <Kbd mac="Command+B" windows="Control+B" linux="Control+B" />

- Display Secondary Sidebar: <Kbd mac="Command+Control+B" windows="Control+Alt+B" linux="Control+Alt+B" />
